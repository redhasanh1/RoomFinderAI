Accessibility Statement

Last Updated: January 24, 2026

Our Commitment

RoomFinderAI is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying relevant accessibility standards.

Conformance Goals

We aim to conform with WCAG 2.1 Level AA standards, which include:

  • Perceivable: Information presented in ways users can perceive
  • Operable: Interface components and navigation are operable
  • Understandable: Information and operation are understandable
  • Robust: Content compatible with assistive technologies

Accessibility Features

Keyboard Navigation

  • All interactive elements accessible via keyboard
  • Visible focus indicators on all clickable elements
  • Logical tab order throughout pages
  • Skip navigation links

Screen Reader Compatibility

  • ARIA labels on interactive elements
  • Semantic HTML structure
  • Alt text for images and icons
  • Descriptive link text
  • Form labels properly associated with inputs

Visual Design

  • Color contrast ratios meet WCAG AA standards
  • Text resizable up to 200% without loss of functionality
  • Clear font choices and sizing
  • No reliance on color alone to convey information

Content Structure

  • Proper heading hierarchy (H1, H2, H3, etc.)
  • Lists and tables used appropriately
  • Clear page titles
  • Consistent navigation across pages

Known Limitations

We are actively working to address the following:

  • Some third-party integrations (Stripe, maps) may have accessibility limitations
  • AI negotiation tool may require screen reader optimization
  • Image uploads lack advanced accessibility features
  • Some complex data visualizations need improvement

Assistive Technologies

RoomFinderAI is designed to be compatible with:

  • Screen readers: NVDA, JAWS, VoiceOver
  • Speech recognition: Dragon NaturallySpeaking
  • Screen magnifiers: ZoomText, MAGic
  • Alternative input devices: Switch controls, eye tracking

Feedback & Assistance

If you encounter accessibility barriers on RoomFinderAI, please let us know:

Contact Accessibility Team:

Email: accessibility@roomfinderai.com

Please include:

  • Description of the accessibility issue
  • Page URL where you encountered the issue
  • Assistive technology you're using
  • Browser and operating system

We aim to respond to accessibility feedback within 5 business days.

Continuous Improvement

We regularly:

  • Conduct accessibility audits
  • Test with assistive technologies
  • Incorporate user feedback
  • Train our development team on accessibility best practices
  • Update this statement as we make improvements

Fair Housing Accessibility

Beyond digital accessibility, RoomFinderAI supports Fair Housing Act requirements:

  • Landlords must provide reasonable accommodations for disabilities
  • Service animals and emotional support animals must be allowed
  • Physical modifications may be permitted at tenant expense
  • See our Legal Center for tenant rights resources
